This Script configures basic Settings for Windows 10.
Tested with Windows 10 Pro, not with Home or Enterprise!

Just run the run.cmd as Administrator!

Some Settings are not disabled, please check your Privacy, Account and Windows Update Settings.
For Account the Account Sync is not disabled.
Disabling distributed Updates for Windows Update are sometimes not disabled.

Wi-Sense needs to be disabled manually in the Wifi Settings

Sometimes App uninstalls do not work.
You need to right click on them in the Startmenu and run Uninstall.
Some Apps are not uninstallable manually.
